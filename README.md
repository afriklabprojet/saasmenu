## 🛠️ **Installation Rapide**

```bash
# 1. Cloner le projet
git clone [repository-url]
cd restro-saas

# 2. Installer les dépendances
composer install
npm install

# 3. Configuration
cp .env.example .env
php artisan key:generate

# 4. Base de données
php artisan migrate

# 5. Démarrer le serveur
php artisan serve
```
